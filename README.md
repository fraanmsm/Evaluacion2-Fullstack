# Sistema de Triage Clínico - Microservicios (Evaluación 3)

Este repositorio corresponde a la evolución y continuidad directa del ecosistema de microservicios desarrollado para la Evaluación 2. Utilizando la misma base arquitectónica, el proyecto se ha extendido para cumplir con los nuevos requerimientos exigidos en la Evaluación 3, integrando capas avanzadas de infraestructura, configuración distribuida y aseguramiento técnico del código original.
